# BrandSync Frontend - Next.js

## Setup Instructions

1. Navigate to frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Open browser to: `http://localhost:3000`

## Features

- **Welcome Screen**: Introduces BrandSync platform and features
- **Chat Interface**: Interactive chat with Brand Research Agent
- **Brand Dashboard**: Shows BrandSync completion and next steps

## Environment Variables

Create `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Axios
- Lucide Icons
